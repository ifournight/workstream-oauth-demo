# 入门指南

本指南将帮助您开始使用 OAuth 演示应用程序。

## 前置要求

- 访问演示应用程序
- 有效的用户账户（需迁移到 Workstream Identity v2）

目前 Identity v2 尚未完全就绪，临时方案如下：

- 注册一个专门用于开发的 users 账号，在 hr.workstream.us 给予相应权限
- 将账号邮箱告知 Workstream 的工程师，将其升级到 Identity v2
- 升级后用户需要重新登录 hr.workstream.us 完成账号注册流程
- 此账号可以用于 oauth apps 的开发

## 概述

OAuth 演示应用程序允许您：

1. **创建 OAuth 客户端** - 为您的应用程序生成客户端 ID 和密钥
2. **获取访问令牌** - 使用客户端凭据交换访问令牌
3. **使用 API** - 使用您的访问令牌进行经过身份验证的 API 调用
