# API 参考

## 身份验证

获得访问令牌后（参见[如何获取访问令牌](./get-token)），您可以使用它通过 Bearer token 身份验证向 Workstream 的公共 API 发送请求。

### Bearer Token 身份验证

在 API 请求的 `Authorization` 请求头中包含访问令牌：

```
Authorization: Bearer <access_token>
```

### 请求示例

```javascript
const response = await fetch('https://api.example.com/protected-resource', {
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'Content-Type': 'application/json',
  },
});
```

## 注意事项

- 访问令牌是 JWT 令牌，在特定时间后过期（通常为 3600 秒 / 1 小时）
- 进行 API 请求时始终使用 HTTPS
- 安全存储令牌并在过期前刷新它们
- 通过获取新令牌来处理 401 未授权响应
